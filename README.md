# Slowdown

Markdown parser written in go. 

> [!NOTE]
> This is project is not being actively developed, I will be back to finish it off (hopefully)
> Billy - March 2025

## Supported Features

+ ATXHeaders (# style headings)
+ Paragraphs
+ Fenced Code Blocks
+ Thematic Breaks

## Not Yet Supported

+ Lists
+ Block Quotes
+ Setext Headings
+ Raw HTML Blocks
+ Link Table
+ Non-fenced code blocks
+ Inline 
	+ Bold
	+ Italics
	+ Inline Code
 	+ Links
	+ Images
	+ Raw HTML
